# React 项目模板

## 技术栈

- React v18
- TypeScript v5
- Vite v5
- Tailwind CSS v3
- ESLint v9
- Ant Design v5

## 创建项目

1. 克隆此仓库：

```bash
npx degit stormzhangbx/react-template <project-name>
cd <project-name>
```

2. 安装依赖：

```bash
pnpm install
```

3. 启动开发服务器：

```bash
pnpm run dev
```

4. 构建项目：

```bash
pnpm run build
```
